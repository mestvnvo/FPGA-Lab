## User’s Guide

### Control Alarm

Left 4 7-segment displays: ALARM TIME in minute.second (MM:SS)  
V10/SW15: On/Off Switch for Alarm  
T15/LED9: Alarm Status (On/Off)  
U11/SW14: Increment Alarm Time  
P17/BTNL: Reset Alarm Count  
N17/BTNC: Reset Alarm and Alarm Sound (Once it has gone off)  
V16/LED8: Alarm Status (Alarm has gone off/Alarm hasn’t gone off)  

### Control Clock

Right 4 7-segment displays: CLOCK in minute.second (MM:SS)  
J15/SW0: System Clock (Off/On)  
L16/SW1: Increment Clock Time  
M17/BTNR: Reset Clock Count

### Requirements:
1. The system shall display the digital clock on four on-board 7-segment displays  
2. The system shall display the set alarm clock using the other four on-board 7-segment displays  
3. The clocks shall be displayed in the format of “minute.second”  
4. The system shall provide an On/Off switch to turn on/off the alarm  
5. The system shall display the alarm status (On or Off) using an LED  
6. When the alarm goes off, the system shall play a song for x period (x is your choice)  
7. The system shall design and provide a user interface for setting up the alarm time  
8. A User Guide shall be provided to the users
